Projects

Here is a handful of projects that I have completed at my time at Epicodus

Five-Star Resort

Five-Star Resort was a project intend for us to learn branching and merging in our git repositories. The idea was to style the same basic page two different ways (using two branches in our git repository) and merge the better looking one into the master branch. This project uses html and css.

Wikipedia Clone

The Wikipedia Clone project was intended for us to get familliar with using Bootstrap. We were to try to mimic the look and feel of the Wikipedia home page using Bootstrap classes. This project uses html, css, and Bootstrap.

Animal Shelter

The Animal Shelter project helped familiarize us with using floats in our css as a way of arranging objects on out page. This project uses html and css.
